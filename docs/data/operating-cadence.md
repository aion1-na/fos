# Operating Cadence

## Weekly

- DUA tracker review.
- Dataset card and access-policy review.
- Schema-change review with contracts owner.
- Backlog triage against DRD traceability.

## Per Sprint

- Confirm no fake data or unversioned reads.
- Confirm each new transform has codebook mapping and quality profile updates.
- Confirm simulation-facing outputs include dataset references.
- Run credential scans and package tests before review.

## Release

- Freeze dataset versions.
- Attach provenance manifests.
- Record access approvals.
- Confirm dashboards and connectors use environment-managed credentials only.
