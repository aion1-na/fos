# Brief Template Language

## Demo Output

This brief is a demo artifact generated from fixtures or request-status metadata. It is not evidence of real-world effect size, causal validity, or policy readiness.

## Research-Grade Output

This brief is research-grade only when dataset references, validation gates, evidence claims, and scientific sign-off logs are complete. Findings should be interpreted with the listed limitations and uncertainty.

## Decision-Grade Output

Decision-grade language is allowed only after approved production data, passing validation gates, complete provenance, and advisor sign-off. Any override must be recorded in the assumptions and audit log.
