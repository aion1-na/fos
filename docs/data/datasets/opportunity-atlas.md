# Opportunity Atlas

Canonical dataset name: `opportunity_atlas`

Access status: request-status stub until public file terms and version are recorded.

License metadata: Opportunity Atlas public data terms and citation requirements must be captured before production ingestion.

Codebook mapping: `codebooks/opportunity_atlas.yaml` maps public mobility outcome aggregates by supported geography and cohort.

Quality profile: production profiles must document geography level, cohort definition, suppression/missingness, uncertainty, and temporal coverage.

Provenance manifest: connector metadata only until public aggregate files are staged with content hash and codebook version.

Access policy: use only public aggregate geography levels supported by the source. No restricted tax microdata is assumed available.

Limitations: mobility aggregates are contextual validation signals, not deterministic predictions for people.

Inappropriate uses: do not rank individuals, infer family income histories, or assign causal effects without a validated design.
