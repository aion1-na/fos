# ILO Public Context

Canonical dataset name: `ilo_public_context`

Access status: request-status stub until indicator selection, license, and vintage are recorded.

License metadata: ILO public-data license and attribution requirements must be captured before production ingestion.

Codebook mapping: `codebooks/ilo_public_context.yaml` maps selected labor and time-use-adjacent indicators into public international context.

Quality profile: production profiles must document indicator definitions, revisions, missingness, country coverage, and comparability limitations.

Provenance manifest: connector metadata only until public indicator extracts are staged with content hashes.

Access policy: public aggregate indicators only.

Limitations: country-level labor context is not a substitute for household, employer, or individual data.

Inappropriate uses: do not infer individual employment outcomes or create causal transition effects from aggregate context.
