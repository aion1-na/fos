# IRB and DUA Checklist

Complete before any restricted health or mortality aggregate ingestion.

- NCHS or state vital statistics access request is approved.
- IRB determination or approval is recorded.
- DUA/license terms permit the intended aggregate output.
- Disclosure-review protocol is recorded.
- Small-cell suppression is applied before export.
- Source environment metadata is attached to every aggregate.
- Public CDC WONDER outputs remain in the public health path, not the restricted path.
- Simulation-facing outputs use `dataset_reference = (canonical_dataset_name, version, content_hash)`.
