# State UI Displacement Analysis Plan

## Purpose

Prepare state unemployment insurance administrative aggregates for job-loss, reemployment, and wage-trajectory calibration.

## Intended Aggregate Outputs

- `state_ui_job_loss_rates_by_industry_region`
- `state_ui_reemployment_latency_distribution`
- `state_ui_wage_trajectory_bins_after_job_loss`

## Required Approvals

- State DUA/license approval.
- IRB determination or approval.
- Legal approval for data handling and publication.
- Disclosure approval for aggregate calibration tables.

## FOS Boundary

No row-level administrative data enters FOS. Administrative-data findings can return only as approved aggregate calibration tables with pinned dataset references.
