# Reading a Brief

Every brief must contain findings, assumptions, uncertainty, evidence trail, validation status, and reproducibility manifest.

## Review Checklist

- Findings are backed by saved finding artifacts.
- Assumptions include overrides and epistemic limits.
- Uncertainty explains sensitivity and seed stability.
- Evidence trail lists cited claim ids.
- Validation status matches the report.
- Manifest identifies the exact run artifact used for reproduction.

Draft runs are rejected before export.
