# Scenarios

A scenario describes a pack, population target, branch set, interventions, shocks, parameters, and stage status.

## Authoring Flow

1. Frame the research question.
2. Compose branches and interventions.
3. Attach evidence claims.
4. Synthesize or select a population.
5. Save a run specification in Configure.
6. Execute the read-only run.
7. Validate claims.
8. Explore subgroup and pathway results.
9. Export a brief only when required sections are complete.

Draft runs use 500 agents, 12 months, 5 seeds, fixture-only evidence, and no validation gates. Draft runs cannot publish briefs.
