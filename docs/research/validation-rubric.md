# Validation Rubric

Validation reports summarize whether a run can support a brief.

## Required Signals

- E-value for headline causal claims.
- Distributional fidelity.
- Seed-stability variance.
- Drift status.
- Evidence-backed pathway attribution.

## Gates

- Green: export allowed.
- Yellow: export allowed only with explicit recorded justification.
- Red: export blocked by default. Override requires privileged written justification of at least 50 characters and is recorded in audit history and brief assumptions.
