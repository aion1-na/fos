RS-03 - Calibrated young-adult population synthesis from ACS, CPS, and GFS marginals.

Objective: Replace seeded/random young-adult state generation with calibrated population synthesis using ACS, CPS, and GFS marginals.

Implement deliverables:
- ACS PUMS and CPS connector hooks for young adults ages 18-29
- Crosswalks for age, education, employment, household, income, geography, and occupation fields
- IPF/raking calibration engine for population marginals
- features.us_young_adult_population_marginals and features.synthetic_population_calibration_diagnostics
- Population card for the flagship synthetic population
- Export format matching current pack.py population state plus dataset_reference provenance

Acceptance gates:
- 5,000-agent population can be generated from real marginals with reproducible seed and full manifest
- Calibration diagnostics report max absolute standardized difference, KL divergence, and marginal coverage
- Population generator refuses to run if ACS/CPS/GFS references are missing unless explicitly in demo mode
- Childhood predictors and current context fields are filled from real distributions or marked as imputed with documented priors

Constraints:
- Do not fabricate data. Fixtures are allowed only for unit tests and must be named fixture_only.
- Every production-facing dataset must have a dataset card, license/access metadata, codebook mapping, quality profile, provenance manifest, and content hash.
- Every simulation-facing output must use dataset_reference = (canonical_dataset_name, version, content_hash).
- MiroFish outputs are interaction/narrative artifacts unless explicitly validated; they must not create causal effect sizes.
- Keep changes scoped to this sprint and do not commit automatically.
